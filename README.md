# SukunaSpam V1.1
SukunaSpam e um SPAM utilizando API de serviços grandes como **Uber**, **99**, **Aiqfome**, **RecargaMulti**, **Vivo** e **Claro**.

## Requisitos
* NodeJS Instalado

## Instalação
* ```apt update && apt upgrade```
* ```apt install git -y```
* ```git clone https://github.com/SukunaBR/SukunaSpam```
* ```cd SukunaSpam```
* ```chmod +x *```
* ```bash install.sh```

## Exemplo de uso
Start: ```node spam.js```
